如何快速的在Windows下用Github和Hexo搭建博客系统

发表于 2017-06-29 | 分类于 Artical

准备

  • 安装 Nodejs。

  • 安装 Git Windows。

    • Github 配置


  • 登录Github，注册一个用户，如xyz。

  • 根据github pages的提示，创建一个仓库名为xyz.github.io， 之后就可以通过 https://xyz.github.io 访问。

    • Hexo 配置


  • 安装Hexo:
    配置好github后，打开git shell，输入npm install hexo-cli -g命令安装。

  • 配置Hexo:
    cd到你自定义用来创建博客的文件夹，输入 npm install hexo --save。

  • 创建Blog：
    hexo init <floder>
    cd <floder>
    npm install

  • 安装插件：
    npm install hexo-generator-index --save
    npm install hexo-generator-archive --save
    npm install hexo-generator-category --save
    npm install hexo-generator-tag --save
    npm install hexo-server --save
    npm install hexo-deployer-git --save
    npm install hexo-deployer-rsync --save
    npm install hexo-deployer-openshift --save
    npm install hexo-renderer-marked@0.2 --save
    npm install hexo-renderer-stylus@0.2 --save
    npm install hexo-generator-feed@1 --save
    npm install hexo-generator-sitemap@1 --save

  • 基本命令：
    1. 创建名为title的新的文章:hexo new "title"，也可在source\posts手动编辑。
    2. 生成本地预览:hexo server, 成功后登录localhost：4000可查看效果。
    3. 生成public静态文件:hexo generate, 会将md文件转成html等文件，public中的文件可发布到github上。
    4. 发布到github:hexo deploy, 将内容发布到github上，前提是需要在config文件中将deploy类型指定为git，仓库为https://github.com/xyz/xyz.github.io。

    • 主题设置


  • 安装主题，可参考Next或Hexo Themes。

  • 在config文件中修改主题，将theme指定到你安装的主题，如Next。

  • 创建tags可输入hexo new page "tags",会在source下生成tags文件夹，打开其中的index.md,添加type: tags即可。

  • 创建categories可输入hexo new page "categories",会在source下生成categories文件夹，打开其中的index.md，添加type： categories即可。
